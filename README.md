# Sophisticated Backpacks Addon

An addon for [Sophisticated Backpacks](https://github.com/P3pp3rF1y/SophisticatedBackpacks) (Minecraft 1.19.2) that adds a "Store All" button to container screens, allowing you to quickly deposit items into your worn or carried backpack.

## Features

- **Store All Button**: Adds a "存入背包" (Store into Backpack) button to most container screens (Chests, Barrels, etc.).
- **Hotkey**: Press `F` while in a container GUI to trigger the store action.
- **Smart Detection**: Automatically finds your backpack in Curios slots or your main inventory.

## Requirements

- Minecraft 1.19.2
- Forge 43.2.0+
- Sophisticated Backpacks
- Curios API

## Development

Built using ForgeGradle 5.1.
